# Unsupervised, Label-Agnostic Feature Selection for Biomedical Data

This repository contains the MATLAB implementation of the paper:

**"Unsupervised, Label-Agnostic Feature Selection for Biomedical Data via Multi-Objective Optimization"**

by Mohammad Hossein Safarpour.

[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.XXXX.svg)](https://doi.org/10.5281/zenodo.XXXX)

## 📌 Abstract

This work presents a novel, label-agnostic, multi-objective feature selection framework for high-dimensional biomedical data. The method jointly optimizes two intrinsic properties: distributional informativeness (via discretized Shannon entropy) and structural coherence (via inverted Davies–Bouldin Index), enhanced by a novel fuzzy crowding distance metric for diversity promotion. It operates with linear complexity O(dn + d log d) and requires only one fixed hyperparameter (K=5).
